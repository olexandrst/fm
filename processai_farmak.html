            Azure OpenAI
          

          
Один ресурс Azure OpenAI може містити декілька deployment'ів різних моделей. Endpoint і Key — спільні, deployment'и — різні.

          

            
            
          

          

            
            
          

          

            
            
          

        


        

          

            
            Модель 1 — Розмова / Інтерв'ю
          

          
Швидка модель для діалогу. Рекомендовано: gpt-4o або gpt-4.1-mini

          

            
            
          

        


        

          

            
            Модель 2 — Структуризація ARIS JSON
          

          
Потужна модель для фінального аналізу. Рекомендовано: gpt-5-mini або gpt-5-chat. Якщо немає доступу — можна дублювати Модель 1.

            Azure Avatar (опціонально)
          

          

            Поточна версія використовує вбудованого SVG-аватара з анімацією. Для підключення фотореалістичного відеоаватара Azure потрібен ресурс з увімкненим Avatar API (Public Preview). Вбудований аватар повністю функціональний.
